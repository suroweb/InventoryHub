# Performance Optimization Skill

This skill analyzes and optimizes application performance, focusing on database queries, caching, and API response times.

## What This Skill Does

1. **Database Query Analysis**
   - Identify N+1 query problems
   - Find missing indexes
   - Detect inefficient joins
   - Analyze query execution plans
   - Suggest query optimizations

2. **Caching Strategy**
   - Identify cacheable data
   - Implement Redis caching
   - Add response caching
   - Set up distributed caching
   - Cache invalidation strategies

3. **API Performance**
   - Measure endpoint response times
   - Identify slow endpoints
   - Add pagination where missing
   - Implement async operations
   - Optimize serialization

4. **Memory Optimization**
   - Detect memory leaks
   - Optimize object allocation
   - Implement object pooling
   - Reduce GC pressure

5. **Blazor WebAssembly**
   - Optimize bundle size
   - Implement lazy loading
   - Add virtualization for lists
   - Optimize JavaScript interop

## Performance Targets

- API response time: <200ms (p95)
- Database queries: <50ms
- Page load time: <2s
- Time to interactive: <3s
- Bundle size: <5MB
